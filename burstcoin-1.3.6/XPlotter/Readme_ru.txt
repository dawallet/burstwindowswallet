XPlotter.exe -id <ID> -sn <start_nonce> [-n <nonces>] -t <threads> [-path <d:\plots>] [-mem <8G>]

<ID> - account ID кошелька
<start_nonce> - Стартовый нонс, по умолчанию = 0
<nonces> - Количество нонсов, по умолчанию = 0
<threads> - Количество потоков, по умолчанию = 1
-path - [Опционально, по умолчанию = текущая папка] Путь к папке с плотом, если папка не существует, то она создается.
-mem <8G> - [Опционально, по умолчанию = 0.5Гб на каждый поток] 

Техническая информация:
Если не указано количество нонсов или оно равно 0, то плот создается на всё свободное место диска.

Вычисление нонсов идет в <threads> потоков. Паралельно вычислениям нонсов (после первой итерации) асинхронно идет запись на диск без использования системного кэширования записи (FILE_FLAG_NO_BUFFERING).

Если свободной ОЗУ не достаточно для формирования нонсов, то количество нонсов на поток уменьшается, количество потоков остается неизменным.

При запуске плоттера сразу создается пустой плот требуемого размера, это предотвращает ошибки записи если по какой-то причине место на диске уменьшится во время работы плоттера.

Запись на диск производится в 4096 итераций примерно по threads*64 КБ.
